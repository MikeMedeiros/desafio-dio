# Receita Hot dog 

## Lista de ingredientes

- Pão 
- salscicha
- molho de tomate
- alho
- cebola
- sal 
- pimenta
- manteiga

## Opcional 

- batata palha
- ketchup
- mostarda

## Preparo 

Pique a cebola e o alho, refogue bem na manteiga.

Acrescente o extrato de tomate, acrescente a  salsicha e deixe cozinhar a gosto.

E é so servir.

Acrescente batata palha, catchup e a mostarda.
